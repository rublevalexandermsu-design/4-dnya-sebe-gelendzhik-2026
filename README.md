# Gelendzhik Retreat Landing

One-page retreat landing for **"4 ДНЯ СЕБЕ"** in Gelendzhik.

## Local preview

- Run: `npm install`
- Start preview: `npm run dev`
- Local host for comparison: `http://127.0.0.1:3037`

## PDF brochure

The brochure is generated to `public/retreat-brochure.pdf` during build and is linked from the hero and pricing sections.

## GitHub Pages

The project is configured for static deployment via `.github/workflows/deploy.yml`.
